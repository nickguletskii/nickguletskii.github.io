GLXOSD

GLXOSD is an OSD/overlay for OpenGL applications running under Linux.

TSLK is an unfinished rewrite of my old project. It is a small educational language that was built to be translatable.

OpenOlympus

OpenOlympus is a work-in-progress opensource programming olympiad hosting server. It consists of 3 parts: OpenOlympus (the web server), OpenOlympus Cerberus (the testing API) and OpenOlympus Olrunner (libsandbox frontend).
